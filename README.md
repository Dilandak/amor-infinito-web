## 💌 Sobre el proyecto

Desde 2022 hasta hoy — 4 años de momentos, risas, viajes, conciertos y recuerdos que no quería que se perdieran en una galería del celular.

**Cristal App** es una aplicación web hecha completamente a mano para mi novia Cristal. Un álbum digital interactivo con fotos organizadas por año, música de fondo, y una experiencia pensada para hacerla sonreír cada vez que la abra.

> *"Sé que amas los recuerdos y las fotos… aquí está un pedacito de lo nuestro."*

---

## ✨ Características

- 💖 **Onboarding animado** — pantalla de bienvenida con corazón interactivo y animaciones GSAP
- 🎵 **Reproductor de música** — canciones de YouTube integradas, con controles de volumen y progreso
- 📸 **Galería de fotos** — organizada por año (2022 → 2026) con modal de detalle y navegación
- 🗓️ **Historia por años** — cada año tiene su título, descripción y sus fotos especiales
- 💌 **Sección "Para ti"** — fotos solo de ella, porque se lo merece
- 💑 **Sección "Nosotros"** — los dos juntos, para siempre
- 🫧 **Corazones flotantes** — porque sí
- 📱 **100% responsive** — se ve igual de bonito en celular y en PC

## 🛠️ Stack tecnológico

| Tecnología | Uso |
|---|---|
| React 18 + TypeScript | Frontend principal |
| Vite | Bundler y dev server |
| Tailwind CSS | Estilos y diseño responsive |
| GSAP | Animaciones fluidas |
| shadcn/ui | Componentes de UI |
| YouTube IFrame API | Reproductor de música |

## 🚀 Correr el proyecto localmente

```bash
# Clonar el repositorio
git clone https://github.com/tu-usuario/cristal-app.git
cd cristal-app

# Instalar dependencias
pnpm install

# Iniciar en desarrollo
pnpm run dev

# Build para producción
pnpm run build
```

## 📁 Estructura del proyecto

```
src/
├── components/
│   ├── gallery/        # Galería de fotos con modal
│   ├── layout/         # FloatingHearts, navbar
│   ├── music/          # MusicPlayer con YouTube API
│   ├── onboarding/     # Pantalla de bienvenida animada
│   └── year-section/   # Historia organizada por año
├── data/
│   ├── photos.ts       # Fotos de ella
│   ├── couple-photos.ts# Fotos de los dos
│   ├── songs.ts        # Lista de canciones
│   └── years.ts        # Datos por año
└── assets/
    ├── 2022/
    ├── 2023/
    ├── 2024/
    ├── 2025/
    └── 2026/
```
